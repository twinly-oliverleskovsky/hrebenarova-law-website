# JUDr. Iveta Hrebenárová - Advokátska kancelária

Moderná, prémiová webová stránka pre advokátsku kanceláriu v Prešove.

## Technológie

- **Next.js 14** - React framework s App Router
- **TypeScript** - Typová bezpečnosť
- **Tailwind CSS** - Utility-first CSS framework
- **Framer Motion** - Animácie a prechody
- **Lucide React** - Ikony

## Inštalácia

```bash
# Nainštalovať závislosti
npm install

# Spustiť vývojový server
npm run dev

# Vytvoriť produkčný build
npm run build

# Spustiť produkčný server
npm start
```

## Štruktúra projektu

```
src/
├── app/
│   ├── globals.css      # Globálne štýly
│   ├── layout.tsx       # Root layout s metadátami
│   └── page.tsx         # Hlavná stránka
└── components/
    ├── Navigation.tsx   # Navigácia s mobilným menu
    ├── Hero.tsx         # Hero sekcia
    ├── About.tsx        # O mne sekcia
    ├── Services.tsx     # Právne služby
    ├── WhyMe.tsx        # Prečo si vybrať mňa
    ├── Process.tsx      # Proces spolupráce
    ├── Contact.tsx      # Kontaktný formulár
    ├── Footer.tsx       # Pätička
    ├── CookieConsent.tsx # GDPR cookie banner
    └── index.ts         # Export komponentov
```
